# 🚀 Landing Page - E-commerce de Productos Digitales

Landing page moderna y optimizada para conversión, diseñada para la venta automatizada de un Mega-Pack de Prompts de IA.

## 🛠️ Tecnologías y Diseño
- **HTML5 & Tailwind CSS** (vía CDN).
- **UI/UX:** Diseño responsivo, modo oscuro y efectos neón/blur.
- **Integración:** Conexión con pasarela de pago y lead magnet (Gumroad).

## 🎯 Funcionalidad
Este proyecto demuestra la capacidad de crear una interfaz web orientada a ventas de un producto digital real, con estructura persuasiva y llamadas a la acción (CTAs) claras.
